Программа для пульта, работающего в составе измерительной стойки.
Пульт позволяет определять длительность, период импульса, поступающего на
его вход 1, а также задержку между фронтом импульса по входу 1 и фронтом импульса
по входу 2. Запус измерения производиться по фронту или спаду импульса по входу 3.
Вывод результата производиться на 4 символьных индикатора, отображающих 4 
старших разряда полученного числа (с запятой), 3 светодиода, для 
автоматического отображения единицы величины, выводимого индикатором числа: «нс», «мкс», «мс»
и на 50-ти контактный разъем в двоично-десятичном коде (до 300 мкс)
